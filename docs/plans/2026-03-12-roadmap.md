## v0.9.0 — Quality & Guard (2-3 weeks)

**Goal:** Physics guard + autoexp pilot. "환각 방지" 기초.

### Physics Guard (Lite)

| Module | Purpose |
|--------|---------|
| `guard/rules.py` | Domain rules: colormap validity, scalar range, vector direction |
| `guard/validator.py` | Post-render validation: pass/warn/fail |
| `tools/validate_render.py` | `validate_render` MCP tool |

Rules examples:
- Pressure → diverging colormap, zero-centered
- Velocity magnitude → sequential colormap, non-negative
- Temperature < 0K → warning
- Empty isosurface → suggest value from data range
- Camera outside bounds → auto-reset

### Autoexp Pilot

| Module | Purpose |
|--------|---------|
| `engine/quality.py` | Render quality metric (contrast, edge entropy, field coverage) |
| `engine/presets.py` | Domain preset library (Git Ratcheting) |
| `core/autoexp.py` | Modify → render → measure → keep/revert loop |

### Quality Gates

- Coverage ≥ 90% (up from 80%)
- CI test count guard: 1550 → match actual
- Benchmark suite in CI (render time regression)
- mypy strict mode on new modules
